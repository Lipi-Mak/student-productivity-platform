# StudentOS - Student Productivity Platform

StudentOS is a full-stack student productivity platform built with Flask that helps students organize their academic life in one place. It combines assignment tracking, attendance management, study planning, semester goal tracking, notes, and timetable management into a clean, responsive dashboard designed for everyday use.

The goal of StudentOS is simple: instead of juggling multiple apps and notebooks, students can manage their entire semester from a single dashboard.

---

## Features

### Authentication
- Secure user registration and login
- Password hashing
- Session management using Flask-Login

### Dashboard
- Personalized welcome page
- Attendance overview
- Today's timetable
- Upcoming assignments
- Study planner checklist
- Semester goals tracker
- Quick notes overview

### Assignment Manager
- Add assignments
- Edit assignments
- Delete assignments
- Due date tracking

### Attendance Tracker
- Add subjects
- Record attended and total classes
- Attendance percentage calculation
- Update and delete records

### Study Planner
- Create study sessions
- Mark tasks as completed
- Edit existing plans
- Delete plans
- Quick add from dashboard

### Semester Goals
- Create semester goals
- Track progress
- Automatic completion handling
- Edit and delete goals

### Quick Notes
- Create notes
- Edit notes
- Delete notes
- Dashboard preview

### Timetable
- Weekly timetable management
- Edit schedule
- Delete schedule entries
